## Saan Bukas?
                                 
This cycling companion app provides a random picker where to go for your future ride. It provides random picker and push notifications so you won’t miss your future bike ride with friends.

## Built With
* UIKit 
* Realm Swift 
* UserNotifications 
* Cocoapods 
* SWIFT
* Storyboards
